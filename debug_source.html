  • To search for a word, e.g., Wille, enter it in the search field.

    A question mark can be inserted in place of a single letter, so W?lle, for example, retrieves Wille but also Wolle or Welle.

    An asterisk can be inserted in place of one or more letters: the search for Wille*, for example, retrieves Wille but also Willens and Willensfreiheit.

    The „Case Sensitive“ checkbox limits the search to words that match the case of  the search term entered. For example, if you search for wille* and select „Case sensitive“, you would retrieve willenverneinende but not Willensfreiheit.

  • You can search for two or more words, e.g. Wille and Macht. Selecting „All these words“ retrieves only passages that contain all the search terms; selecting „At least one of these words“ retrieves passages that contain at least one of the terms entered.
  • Quotation marks constrict the search to the exact phrase entered, e.g., „Wille zur Macht.“
  • The + button adds additional constraints on the search, including:

    Time period, e. g., only occurrences in texts that were written in 1872 or only texts written in August 1880.

    Group of texts, e. g., only occurrences in published works or only occurrences in posthumous fragments or only occurrences in Die Geburt der Tragödie.

    It is also possible to limit the search to a group of letters, e. g. only the letters to Carl von Gersdorff written from Bonn.

    By adding new search facets, you can combine different search options. For instance, you can search the expression Macht in the period between 1884 and 1888 and you can limit this search to letters and posthumous fragments.
