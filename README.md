# Nietzsche-KR-Translation

# 무엇을 하는 레포인가요?
http://www.nietzschesource.org/#eKGWB
독일 철학자 프리드리히 니체(Friedrich Nietzsche)의 모든 1차자료 독일어 텍스트를 한국어로 번역하는 레포입니다. 현재 계획은 다음과 같습니다.
- 1차자료 텍스트를 Markdown으로 정리
- 텍스트를 AI번역
- 번역 수정(어색한 표현 수정 등)

## 참고 자료
