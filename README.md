# Nietzsche-KR-Translation

## 무엇을 하는 저장소인가요?

프리드리히 니체(Friedrich Nietzsche)의 모든 1차자료 독일어 텍스트를 한국어로 번역하는 저장소입니다. 
개인 공부용으로 만든 저장소입니다. 
현재 http://www.nietzschesource.org/#eKGWB 의 모든 텍스트는 복사가 완료되었습니다.

### 5. 노트 유고 (Nachgelassene Fragmente)
노트 유고는 연도와 그룹 번호로 구분됩니다.
형식: `NF-년도, Gruppe 번호` (예: `NF-1869, 1`)
약어: **NF**

### 6. 편지 (Briefe)
편지는 연도별로 분류됩니다.
형식: `BVN-년도` (예: `BVN-1850`)
약어: **BVN** (Briefwechsel)
